## Getting Started

### Prerequisites
- Python 3.x

### Installation
1. Clone the repository:
  ```bash
  git clone https://github.com/T-0n-1/darts_competition_result_service.git
```
2. Navigate to the project directory:
  ```bash
  cd darts_competition_result_service
```
### Running the Application
1. Run the main script to manage and report darts competition results:
  ```bash
  python ratkaisu.py
```

## Key Features
